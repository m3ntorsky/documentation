# RegisterRawAlias
Registers a raw alias (without sw_ in console) for a command registered through Swiftly Commands.
::: tabs
@tab Lua
```lua
@returns void
commands:RegisterRawAlias(commandName --[[ string ]], alias --[[ string ]])
```

:::
