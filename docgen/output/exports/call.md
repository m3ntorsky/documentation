# Call Export
Calls the specified export.
::: tabs
@tab Lua
```lua
@returns Any* any
exports["PLUGIN_NAME"]:EXPORT_NAME(... --[[ Any* any ]])
```
> [!warning]
> Replace `PLUGIN_NAME` with the plugin name for the specific export.
> Replace `EXPORT_NAME` with the export name.
:::
